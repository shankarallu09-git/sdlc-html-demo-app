# SDLC HTML Demo App

A minimal, pure-HTML demo application for testing the SDLC pipeline.  
No frameworks, no build tools — just HTML + CSS + JS in a single file.

## Apps

| File | Description |
|------|-------------|
| `index.html` | Task Tracker — add, complete, and delete tasks with local storage |
| `counter.html` | Counter App — increment/decrement with configurable step size |

## How to Run

Simply open `index.html` or `counter.html` in any browser. No server needed.

## Purpose

This app is intentionally simple to speed up SDLC demo cycles:
- Instant "code generation" (it's already written)
- Fast code review (minimal code)
- Quick test execution
